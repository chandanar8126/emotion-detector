# Real-Time Facial Emotion Detector

A real-time facial emotion detection system built with Python, OpenCV and DeepFace.

## Features
- Detects 7 emotions: Happy, Sad, Angry, Surprise, Fear, Disgust, Neutral
- Live emotion analytics panel with percentage bars
- Emotion timeline graph
- Auto screenshot on strong emotion detection
- Multithreaded for smooth real-time performance

## Technologies Used
- Python, OpenCV, DeepFace, NumPy, Threading

## How to Run
pip install opencv-python deepface tf-keras numpy
python emotion_detector.py

## Controls
- S = Save screenshot manually
- Q = Quit
